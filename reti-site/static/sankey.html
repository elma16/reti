What This Diagram Counts

Each node is one FCE ending row, including the auxiliary pawnless and connected-pawn subrows used in the table. For a selected corpus and half-move threshold, each game contributes the endings whose first run meets that threshold.

Within a game, those qualifying endings are ordered by the ply where their first marker appears. A link counts a consecutive transition from one first-marker stage to the next. If several endings first appear on the same ply, they are treated as co-present at the same stage; the diagram does not invent an order between them.

The counts are link incidences, not mutually exclusive game counts. A single game can contribute several consecutive links if it passes through several different FCE endings. Exact counts and percentages are shown in link hover text rather than directly on the flows.
