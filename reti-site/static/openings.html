What This Page Counts

Select one ECO base code, such as B90, and the table shows how often each FCE ending appears in games with that opening code. The denominator for Opening corpus % is the exact number of source games with that ECO base in the selected corpus.

The counts use the same first-run semantics as the main table: each game can count once for each ending stem if the first run for that stem meets the selected half-move threshold. Tablebase columns and example games are intentionally omitted here.
