Welcome to CQL 6.2

For full documentation, see the CQL website. [For alpha releases, change "/cql" to "/alpha"] This distribution contains:
  • cql.exe, the Windows executable (on Windows)
  • cql, the Mac executable (on Mac)
  • The examples/ subdirectory, which contains example CQL files. These examples are also available from the CQL website by clicking the Examples link in the sidebar.
  • ASCII versions of all the examples, in examples-ascii/.
  • A brief list of Unicode symbols, in palette.cql.
  • The file sample.pgn containing a selection of studies from the Heijden database of endgame studies. For more information about sample.pgn, see sample.pgn description
  • A font name CqlChessMonox.ttf where the symbol x, if present, is a font version number.
Additional installation instructions are available from the installation link located on the website's sidebar.
